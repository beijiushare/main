# 多层视差背景配置说明

## 文件位置

`public/backimg/data.json` — 主页全屏背景的图层配置

## 顶层结构

```json
{
  "type": "multi-layer",
  "multiLayer": {
    "version": 1,
    "layers": [  ]
  }
}
```

- `type` — 固定为 `"multi-layer"`
- `multiLayer.version` — 版本号（当前为 1）
- `layers` — 图层数组，**从远到近**排列（索引 0 = 最远，末尾 = 最近）

## 图层对象

```json
{
  "src": "backimg/xxx.png",
  "x": 0,
  "y": 0,
  "scale": 1,
  "zoom": 1,
  "angle": 0,
  "distance": 0
}
```

## 属性详解

### src

图片路径，相对于 `public/` 目录。

### x, y — 初始位置

图层中心在屏幕上的位置，以**屏幕中心为原点**的归一化坐标：

| 值 | 含义 |
|----|------|
| `0, 0` | 屏幕正中心 |
| `0.5, 0` | 屏幕右 1/4 处 |
| `-0.5, 0` | 屏幕左 1/4 处 |
| `0, 0.5` | 屏幕下 1/4 处 |

范围 -1 ~ 1，`-1` = 屏幕边缘，`0` = 中心，`1` = 另一侧边缘。

### scale — 初始缩放

鼠标在屏幕中心时图层的缩放倍数。

- `1` = 图片高度填满视口高度
- `0.5` = 图片高度为视口高度的一半
- `2` = 图片高度为视口高度的两倍

### zoom — 目标缩放

鼠标移到屏幕边缘时图层的缩放倍数。

缩放值随鼠标距离中心**线性变化**：

```
当前缩放 = scale + (zoom - scale) × 鼠标距离
```

例如 `scale: 1, zoom: 1.2`，鼠标在中心时图 1x，移到边缘时放大到 1.2x。

### angle — 移动角度偏移（度）

控制图层随鼠标移动的方向偏移：

| angle | 效果 |
|-------|------|
| `0` | 图层与鼠标同向移动 |
| `90` | 图层垂直于鼠标方向移动 |
| `180` | 图层与鼠标反向移动 |
| `-90` | 图层反向垂直移动 |

相当于：**图层移动方向 = 鼠标方向 + angle**

### distance — 最大位移量

鼠标移到屏幕边缘时，图层从初始位置偏移的最大距离（归一化坐标，0 ~ 1）。

| distance | 效果 |
|----------|------|
| `0` | 静止不动 |
| `0.3` | 边缘时偏移半个屏幕的 30% |
| `1` | 边缘时偏移半个屏幕 |

位移随鼠标距离中心**线性变化**：

```
位移量 = distance × 鼠标距离
```

```
最终位移 X = cos(鼠标方向 + angle) × distance × 鼠标距离 × 视口宽/2
最终位移 Y = sin(鼠标方向 + angle) × distance × 鼠标距离 × 视口高/2
```

注意 `distance` 不同于之前的 `speed`——`distance` 直接表示"最大能飘多远"，而不是速度倍率。

## 核心逻辑

鼠标行为由一个参数 `t` 控制一切：

```
t = 鼠标距屏幕中心的距离（0 → 1）
  0 = 鼠标在中心（静止）
  1 = 鼠标在边缘（最大效果）
```

各属性随 `t` 的变化：

| 属性 | 公式 |
|------|------|
| 当前缩放 | `scale + (zoom - scale) × t` |
| 位移 X | `cos(鼠标角度 + angle) × t × distance × 视口宽/2` |
| 位移 Y | `sin(鼠标角度 + angle) × t × distance × 视口高/2` |

## 快速示例

### 纯静态背景

```json
{
  "src": "backimg/bg.png",
  "x": 0, "y": 0,
  "scale": 1, "zoom": 1,
  "angle": 0, "distance": 0
}
```

### 跟随鼠标的前景元素

```json
{
  "src": "backimg/flower.png",
  "x": 0.3, "y": 0.2,
  "scale": 0.6, "zoom": 0.8,
  "angle": 0, "distance": 0.3
}
```

### 反向漂移的远景

```json
{
  "src": "backimg/cloud.png",
  "x": -0.2, "y": -0.3,
  "scale": 0.8, "zoom": 0.8,
  "angle": 180, "distance": 0.1
}
```

## 图片素材建议

所有图片宽度统⼀ **1920px**，PNG 透明底（背景层可⽤ JPG/WebP），横版构图。图片⾼度按实际内容裁切，不限。
